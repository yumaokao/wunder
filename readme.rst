wunder
======

Introduction
------------
* `wunder` is a command line interface of WunderList.
* features:
  + command line interface 


Wanted
------
* Wanted features
  + ncurses interface


Build Up
--------
* initial a npm package
  code:: sh

  $ npm init
  $ node index.js

* npm install
  code:: sh

  $ npm install --save commmander
  $ npm install --save wunderlist


Debug
-----
* node-inspector
  code:: sh

  $ sudo npm install -g node-inspector
  $ node-debug index.js
